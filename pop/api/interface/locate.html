public interface locate

\brief Find closest values in a monotonic series, returns the indexes. \details Given an array x(1:n), and given a value y, returns a value j such that y is between x(j) and x(j+1).\n x must be monotonically increasing.\n j=0 or j=N is returned to indicate that x is out of range. \param[in] "real(dp/sp) :: x(:)" Sorted array \param[in] "real(dp/sp) :: y[(:)]" Value(s) of which the closest match in x(:) is wanted \return integer(i4) :: index[(:)] — index(es) of x so that y is between x(index) and x(index+1) \note x must be monotonically increasing.\n \author Matthias Cuntz \date May 2014
