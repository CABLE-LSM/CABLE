public interface linspace

\brief Evenly spaced numbers in interval. \details Return N evenly spaced numbers over a specified interval [lower,upper]. \f[ linspace(lower,upper,N) = lower + arange(0,N-1)/(N-1) * (upper-lower) \f] Output array has kind of lower. \param[in] "integer(i4/i8)/real(sp/dp) :: lower" Start of interval. \param[in] "integer(i4/i8)/real(sp/dp) :: upper" End of interval. \param[in] "integer(i4) :: nstep" Number of steps. \return kind(lower) :: linspace(N) — 1D array with evenly spaced numbers between lower and upper. \authors Matthias Cuntz \date Jun 2016
