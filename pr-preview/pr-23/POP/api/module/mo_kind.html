\file mo_kind.f90 \brief Define number representations \details This module declares the desired ranges and precisions of the number representations, such as single precision or double precision, 32-bit or 64-bit integer, etc. It is similar to the nrtype module of Numerical Recipes in f90. \authors Matthias Cuntz, Juliane Mai, Nov 2011 \date 2011-2017
