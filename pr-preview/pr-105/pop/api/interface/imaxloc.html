public interface imaxloc

\brief First index location in an array of the element with the maximum/minimum value. \details Fortran intrinsics maxloc and minloc return arrays with all indexes corresponding to the maximum/minimum value in an array.\n This routine returns only the first entry as scalar integer. \param[in] "integer(i4/i8)/real(sp/dp)/complex(spc/dpc) :: array(:)" Input array \param[in] "logical :: mask(:)" If present, only those locations in array corresponding to the true values in mask are searched for the maximum value. \return integer(i4) :: imaxloc/iminloc — First index location of maximum/minimum \authors Matthias Cuntz, Juliane Mai \date Feb 2014
