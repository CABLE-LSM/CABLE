The SUBROUTINE ruff_resist is structured as:

  • evaluation of the canopy height and leaf area given the presence of snow (or not)

  • sets the value of soil and snow roughness lengths (depends on configuration of CABLE)

  • evaluates the remaining output variables, depending on whether the land point is vegetated or not.

  • if the soil model used is LSI - updates the evaluated %rt0us
